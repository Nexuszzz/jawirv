Instructions:
=============

1. Open JAWIR CLI in a separate terminal:
   cd D:\expo\jawirv3\jawirv2\jawirv2\backend
   python jawir_cli.py

2. Copy-paste each prompt above one by one

3. Watch for these in the output:
   🔧 ACTION - Shows which tool is being called
   👁️ OBSERVE - Shows the tool result
   ✅ Final response from agent

4. Verify that expected tools are actually called

5. Check if responses make sense and are formatted nicely

Expected Results:
================
- Each scenario should call the expected WhatsApp tool(s)
- Agent should understand the context and combine tools appropriately
- Responses should be in Javanese style with emojis
- No errors from GoWA client (device logged in check should pass)
- Tools should execute successfully and return proper data
